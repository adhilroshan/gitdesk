## ðŸ’» Getting Started

### Prerequisites

- Node.js 18+
- npm / pnpm / bun

### Installation

```bash
# Clone the repository
git clone https://github.com/adhilroshan/gitdesk-io.git

# Navigate into the directory
cd gitdesk-io

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Build

```bash
npm run build
```
